## 策略说明

**复合多因子策略** = 动量(40%) + 低波动(30%) + 量价(30%)

- **动量因子**: 20日涨幅，追涨强势股票
- **低波动因子**: 年化波动率，筛选稳健标的
- **量价因子**: 成交量比率，确认趋势有效性

**调仓频率**: 每月一次
**目标**: 年化收益 25%+，最大回撤控制在 -10% 以内

## 每日报告内容

1. **策略历史表现回顾** - 对比各策略收益/夏普/回撤
2. **当前持仓分析** - QUBT持仓盈亏分析
3. **今日选股推荐** - 基于复合策略的TOP5股票
4. **具体操作建议** - 资金配置方案 (35%/25%/20%/10%/10%)
5. **风险提示** - 止损止盈建议
6. **操作检查清单** - 每日必做事项

## 定时任务配置 (已配置)

```bash
# 每天早上8点生成复合策略日报
openclaw cron add \
  --name "morning-stock-report-v2" \
  --cron "0 8 * * *" \
  --exact \
  --session isolated \
  --message "执行 daily_report.py 生成基于复合多因子策略的美股日报" \
  --description "每日8点复合策略美股日报" \
  --announce --channel feishu

# 每天凌晨3点更新数据
openclaw cron add \
  --name "midnight-stock-update" \
  --cron "0 3 * * *" \
  --exact \
  --session isolated \
  --message "执行 update_stock_data.py 更新美股数据缓存" \
  --description "每日凌晨3点美股数据更新" \
  --no-deliver
```
